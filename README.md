# Mern-Projects

A showcase of full-stack web applications built with the MERN stack and modern web technologies — all deployed to production.

**Live:** https://mern-projects.netlify.app

---

## Projects

| Project | Description | Stack | Live |
|---|---|---|---|
| **N-Pass** | Local-first password manager with generator & strength meter | React, Vite, Tailwind | [vercel.app](https://n-pass-navy.vercel.app) |
| **iDO** | Priority-based todo app with live stats and filter tabs | React, Vite, Tailwind | [vercel.app](https://i-do-rouge.vercel.app) |
| **DeLinks** | URL shortener with custom aliases, no signup required | Next.js, MongoDB, Tailwind | [netlify.app](https://delinks-0.netlify.app) |
| **LinkTree Clone** | Full-stack Linktree clone with custom handles and link management | Next.js, Prisma, PostgreSQL, Supabase | [netlify.app](https://glistening-tarsier-b0cc69.netlify.app) |
| **Get Me A Chai** | Patreon-inspired creator support platform with GitHub OAuth and Razorpay payments | Next.js, NextAuth, Prisma, Neon DB | [netlify.app](https://patreon-c.netlify.app) |

---

## Tech Stack

- **Frontend** — React, Next.js, Vite, Tailwind CSS
- **Backend** — Next.js API Routes, Node.js
- **Databases** — MongoDB Atlas, PostgreSQL (Supabase, Neon)
- **Auth** — NextAuth.js (GitHub OAuth)
- **Payments** — Razorpay
- **ORM** — Prisma
- **Deployment** — Vercel, Netlify
